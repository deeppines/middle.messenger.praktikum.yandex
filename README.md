## Установка

Для корректной работы потребуется [Node.js](https://nodejs.org/) версии 14 и выше.

```shell
git clone https://github.com/deeppines/middle.messenger.praktikum.yandex.git
```

```shell
npm i
```

```shell
npm run start
```
